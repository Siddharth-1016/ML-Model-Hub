🚀 IntelliPredict – Multi-Dataset Machine Learning Web App

IntelliPredict is an end-to-end Machine Learning web application that allows users to upload datasets, train multiple ML models, compare their performance, and generate predictions — all from a simple interactive interface.

This project is designed as a mini AutoML / ML Playground, helping beginners and recruiters see how ML models move from data → training → evaluation → deployment.

✨ Key Features

🔹 Upload and test multiple datasets
🔹 Supports Regression & Classification problems
🔹 Train multiple ML models with one click
🔹 Compare models using performance metrics
🔹 Download trained models for reuse
🔹 Simple and interactive web interface
🔹 Built completely from scratch using VS Code
